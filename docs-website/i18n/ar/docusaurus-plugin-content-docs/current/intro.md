# نظام إدارة المحتوى القرآني - وثائق المطورين

**واجهة Itqan CMS API** هي واجهة REST للوصول إلى بيانات تلاوات القرآن الكريم — القراء، التلاوات، الروايات، القراءات، وتوقيتات الآيات الصوتية الكاملة — من ناشرين موثَّقين.

معرّفات الموارد أعداد صحيحة ثابتة — آمنة للحفظ والإشارة والبناء عليها عبر إصدارات الـ API.
