# المصادقة

واجهات برمجة تطبيقات Itqan مفتوحة حاليًا للجميع — ابدأ التكامل فورًا دون أي إعداد أو بيانات اعتماد أو ترويسات مطلوبة.

تجربة مصادقة مخصصة في الطريق. نصمّمها لتبقي عملية الإعداد سهلة، مع فتح الباب أمام ميزات أكثر ثراءً تعتمد على الحسابات.

سيتم الإعلان عن مزيد من التفاصيل فور الاستعداد. ستستمر عمليات التكامل الحالية في العمل دون انقطاع.
