# Authentication

Itqan APIs are currently open to everyone — start integrating right away with no setup, no credentials, no headers required.

A dedicated authentication experience is on the way. We're designing it to keep onboarding simple while opening the door to richer, account-aware features.

Further details will be announced once it's ready. Existing integrations will continue to work without disruption.
