# Frontend (Angular 19)

- CSR now (Signals, Vite); SSR later via Angular Universal.
- Auth: Auth0 SPA SDK.
- UI: NG-ZORRO.

## Dev
- Node 20+, pnpm or npm
- Env: copy `.env.example` to `.env`

### Scripts (suggested)
- dev: `npm run dev`
- build: `npm run build`
- test: `npm test`
- lint: `npm run lint`

## Notes
- Follow Prompt Structure Protocol in `.cursor/rules/cms-v1.mdc`.
- API base URL comes from env.
